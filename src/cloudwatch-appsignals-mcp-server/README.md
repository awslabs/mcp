# cloudwatch-appsignals-mcp-server (Deprecated)

> **This package is deprecated.** Please use
> [`awslabs.cloudwatch-applicationsignals-mcp-server`](https://awslabs.github.io/mcp/servers/cloudwatch-applicationsignals-mcp-server/)
> instead.

## What happened?

This package (`awslabs.cloudwatch-appsignals-mcp-server`) has been renamed to
`awslabs.cloudwatch-applicationsignals-mcp-server`. The new package contains all
the features of this package plus additional tools and improvements.

Installing this package will automatically install the new package and redirect
all commands to it.

## Migration

Update your MCP configuration to replace `cloudwatch-appsignals-mcp-server` with
`cloudwatch-applicationsignals-mcp-server` in both the server key and `args` array.

See the [new package documentation](https://awslabs.github.io/mcp/servers/cloudwatch-applicationsignals-mcp-server/)
for full installation and configuration instructions.
