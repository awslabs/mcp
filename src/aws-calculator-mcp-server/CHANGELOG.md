# Changelog

## 0.1.0 (2026-05-07)

### Features

- Initial release with 159 AWS services supported
- Playwright-based browser automation for calculator.aws
- Autosuggest handling for instance type fields
- Radio button and dropdown selection support
- Dynamic region selection
- Shareable estimate URL generation
- 400 BDD certification scenarios (pytest-bdd)
- Cross-validated pricing accuracy against AWS Pricing Bulk API
