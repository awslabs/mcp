# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

### How to Report

1. **AWS Security**: Report via [AWS Vulnerability Reporting](https://aws.amazon.com/security/vulnerability-reporting/)
2. **GitHub Security Advisories**: [Report privately](https://github.com/awslabs/mcp/security/advisories/new)

### Response Timeline

- Acknowledgment: 48 hours
- Assessment: 1 week
- Fix: Based on severity
